3. BTW

3.1 Linux 进程状态

  • D 不可中断 Uninterruptible sleep(usually IO)
  • R 正在运行，或在队列中的进程
  • S 处于休眠状态
  • T 停止或被追踪
  • Z 僵尸进程
  • W 进入内存交换（从内核 2.6 开始无效）
  • X 死掉的进程

进程状态的修饰：

  • < 高优先级
  • N 低优先级
  • L 有些页被锁进内存
  • s 包含子进程
  • + 位于后台的进程组
  • l 多线程，克隆线程

3.2 tmux 常用操作

  • tmux new -s <name> 新建会话并取个名字。
  • tmux ls 查看所有会话。
  • tmux a -t <name> 进入某个会话。
  • tmux kill-session -t <name> 终止某个会话。
  • Ctrl + B + D 退出某个会话（仍在后台）。
  • Ctrl + B + S 切换到另外的会话。
