问题分析

直觉就是使用 BFS 来解，每次放入只有一个 edge 的 node（现在的 leaf）。然后直到只剩最上面一层。注意考虑单独的 node（和别的 node 不相连）。比如: [[1,2], [2,3]], n = 6 这种情况，就有 [0, 4, 5] 三个点都不和别的 node 相连。还有 [], n = 2 的时候就要返回 [0, 1]。
